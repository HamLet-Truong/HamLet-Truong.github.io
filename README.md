# Personal CV Website

Professional one-page CV website for **Truong Van Long**.

## Project Structure

```
.
|-- index.html
|-- assets/
|   |-- css/
|   |   `-- styles.css
|   |-- js/
|   |   |-- cv-data.js
|   |   `-- main.js
|   `-- data/
```

## Quick Edit Guide

- Update CV content in `assets/js/cv-data.js`.
- Update design and layout in `assets/css/styles.css`.
- Keep `index.html` for semantic structure only.

## Run Locally

Open `index.html` directly in browser, or use any local static server.
